<task type="auto">
  <name>Task 1: Create TenantHistoryTimeline component</name>
  <files>frontend/src/components/TenantHistoryTimeline.tsx</files>
  <action>
Create a new component `TenantHistoryTimeline` that:

1. **Props**: `{ tenantId: string }` — receives the tenant ID to fetch history for.

2. **Data fetching**: Use `useQuery` with key `['tenant-history', tenantId]`:
   ```typescript
   const { data, isLoading } = useQuery({
     queryKey: ['tenant-history', tenantId],
     queryFn: () => apiClient.get(`/tenants/${tenantId}/history`, { params: { limit: 50 } }).then(res => res.data),
     enabled: !!tenantId,
   });
   ```

3. **Timeline UI**: Build a vertical timeline with colored dots and connector lines:
   - Each event has a colored dot on the left (blue for contract, orange for invoice, green for payment)
   - A vertical line connects the dots
   - On the right of each dot: event title, date (formatted), and key data summary
   - Use Tailwind-style classes: `border-l-2 border-muted` for the timeline line
   - Color dots: `bg-blue-500` for contract, `bg-orange-500` for invoice, `bg-green-500` for payment

4. **Event data display**:
   - Contract: show room name, rent price, status badge
   - Invoice: show total amount, payment status, billing period
   - Payment: show amount, payment method, invoice number

5. **Empty state**: Show a centered message with a Clock icon: `t('tenants.history.noHistory')`

6. **Loading state**: Show 3 skeleton pulse placeholders (simple `animate-pulse` divs)

7. **Icons**: Use lucide-react icons:
   - `FileText` for contracts
   - `Receipt` for invoices
   - `Banknote` for payments

8. Use `useTranslation()` for all text labels. Key format: `tenants.history.*`

9. Format dates using `toLocaleDateString()` — respect the current locale.

10. Format currency amounts using a simple formatter: `amount.toLocaleString()` + ` đ`.
